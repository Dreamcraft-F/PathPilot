## 一、项目概述

PathPilot 是一款 Windows 资源管理器隐形导航员，核心功能是无感记录用户在资源管理器中访问的路径，并提供快速跳转。

**技术栈**: Python 3.11+ / PySide6 / PySide6-Fluent-Widgets / SQLite / pywin32 / comtypes  
**Conda 环境**: `pysfm`  
**启动方式**: `conda activate pysfm && python src/main.py`

## 四、关键约束

- **GUI 层不修改业务数据**，只发信号
- **跨线程操作必须通过 Qt 信号**
- **数据库操作必须加锁** (`self._lock`)
- **不引入新的大型依赖**

## 五、下一步工作建议

### 功能扩展

1. **路径分组显示** — 按应用类型自动分类路径（文档、代码、媒体等）
2. **搜索功能** — 全局路径搜索
3. **快捷键支持** — 全局热键打开悬浮球/主窗口
4. **路径统计** — 访问频率图表、热力图
5. **云同步** — 多设备同步路径记录
6. **自动更新** — 检查新版本并自动下载

### 技术优化

1. **减少打包体积** — 进一步裁剪 PySide6 模块
2. **性能优化** — 大量路径时的渲染性能
3. **内存优化** — 长时间运行的内存占用

### 待修复

1. 深色主题支持 — 当前已移除，可重新实现
2. Windows 10 兼容性测试 — DWM API 行为可能不同

## 七、重要文件说明

| 文件 | 作用 |
|------|------|
| `src/main.py` | 入口，BASE_DIR 计算，首次运行检测 |
| `src/app.py` | 应用主类，所有信号连接 |
| `src/core/window_monitor.py` | 核心监控逻辑，0.5s 轮询 |
| `src/core/path_engine.py` | 引擎协调器，事件队列 |
| `src/gui/floating_ball.py` | 悬浮球，含 DWM 修复 |
| `src/gui/main_window.py` | 主窗口，左侧导航 |
| `src/database/db_manager.py` | 数据库操作，线程安全 |
| `config/default_config.json` | 默认配置 |
