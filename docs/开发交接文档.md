## 一、项目概述

PathPilot 是一款 Windows 资源管理器隐形导航员，核心功能是无感记录用户在资源管理器中访问的路径，并提供快速跳转。

**技术栈**: Python 3.11+ / PySide6 / PySide6-Fluent-Widgets / SQLite / pywin32 / comtypes  
**Conda 环境**: `pysfm`  
**启动方式**: `conda activate pysfm && python src/main.py`

## 四、重要技术细节

### 1. Windows 11 悬浮球圆形修复
Windows 11 会为窗口叠加圆角背景（Backdrop），必须通过 DWM API 关闭：

```python
# src/gui/floating_ball.py
def _fix_win11_window(hwnd):
    ctypes.windll.dwmapi.DwmSetWindowAttribute(hwnd, 33, ctypes.byref(ctypes.c_int(1)), 4)  # 关闭圆角
    ctypes.windll.dwmapi.DwmSetWindowAttribute(hwnd, 38, ctypes.byref(ctypes.c_int(1)), 4)  # 关闭系统背景
    ctypes.windll.dwmapi.DwmSetWindowAttribute(hwnd, 34, ctypes.byref(ctypes.c_int(0)), 4)  # 去掉边框颜色
```

### 2. Nuitka 打包后路径检测
Nuitka 打包后 `__file__` 不存在，使用 `sys.argv[0]` 获取 exe 路径：

```python
# src/main.py
BASE_DIR = os.path.dirname(os.path.abspath(sys.argv[0]))
```

### 3. 开机自启动实现
使用启动文件夹方式（非注册表）：

```python
# src/utils/autostart.py
startup_folder = os.path.join(os.environ['APPDATA'], r'Microsoft\Windows\Start Menu\Programs\Startup')
shortcut_path = os.path.join(startup_folder, 'PathPilot.lnk')

# 创建快捷方式
shell = Dispatch('WScript.Shell')
shortcut = shell.CreateShortCut(shortcut_path)
shortcut.TargetPath = exe_path
shortcut.save()
```

### 4. 日志配置
- 文件日志：INFO 级别，最大 2MB，保留 3 份
- 控制台日志：INFO 级别
