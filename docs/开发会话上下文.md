# PathPilot 开发会话上下文文档

## 📋 项目概述

**PathPilot** 是一款 Windows 资源管理器隐形导航员，用于记录用户在资源管理器中访问的路径，并提供快速跳转功能。

### 核心功能
- 无感记录用户访问的文件夹路径
- 树形目录结构展示历史路径
- 按时间分组展示历史路径
- 一键跳转到历史路径
- 收藏重要路径
- 隐私模式（暂停记录）
