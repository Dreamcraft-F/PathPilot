# PathPilot（资源管理器隐形导航员）原始设计方案

## 1. 开发背景与核心目标

Windows 自带的资源管理器在多任务处理时非常低效：打开新文件夹容易开一堆窗口，或者需要反复点击"后退"按钮，找不到刚才看过的路径。

市面上的替代品（如 Q-Dir、Clover）要么界面太丑，要么强行修改系统资源管理器，导致不稳定。

**我们的目标**：开发一个完全独立的外部工具。它绝对不修改、不注入、不替换 Windows 原生资源管理器。它只做一个功能——在后台默默记录你在原生资源管理器里去过哪里，并在你需要时，帮你一键跳回去。

## 2. 核心原则（开发红线）

1. **独立性**：软件是一个独立的系统托盘程序，不依赖任何系统文件修改。
2. **零侵入**：用户在使用原生资源管理器时，感觉不到这个软件的存在。
3. **只做调度**：软件本身不显示文件列表，不实现文件管理功能（如复制、粘贴、重命名），只管理"路径历史记录"。

## 3. 第一部分：核心框架功能（必须最先实现的基础）

这部分是软件的骨架，必须稳定、极简。

### A. 界面交互：动态三态窗口

#### 状态1（完全隐形）
- 平时软件缩在系统托盘里，不占屏幕空间。

#### 状态2（悬浮球）
- 当主窗口关闭后，在屏幕边缘显示一个极小的半透明悬浮球（直径约16px）。
- 鼠标移开10秒后，它自动消失变回托盘图标。
- 点击悬浮球可唤出主窗口。

#### 状态3（主窗口）
- 点击托盘或悬浮球弹出。
- 这是一个纯文本列表窗口，没有复杂的边框和标题栏。
- 打开时自动聚焦搜索框。
- 如果用户3秒内没有任何操作，主窗口自动收起变回悬浮球。

### B. 无感历史捕获引擎（核心逻辑）

#### 怎么记录
- 软件在后台监听 Windows 事件。
- 当原生资源管理器窗口被激活（获得焦点）时，读取该窗口当前打开的路径。

#### 记录什么
- 只记录路径字符串。

#### 去重逻辑
- 如果用户一直在同一个文件夹里操作，不要重复记录。
- 只记录路径发生变化的时刻。

#### 时效性
- 自动识别"临时路径"（只访问一次）和"高频路径"（连续多天访问）。
- 在列表中通过简单的符号（如 ★）区分。

### C. 路径跳转与窗口调度

#### 点击跳转
用户在 PathPilot 列表中点击某条路径后，软件执行以下判断：

1. **如果该路径已经在某个打开的资源管理器窗口中**：
   - 直接激活并置顶那个窗口（不要新开窗口）。

2. **如果该路径没有打开**：
   - 调用系统命令 `explorer.exe 路径`，用原生资源管理器打开它。

#### 结果
- 用户点击 PathPilot 后，PathPilot 主窗口立刻收起，屏幕上只剩下原生资源管理器。

## 4. 第二部分：完善体验功能（框架搭好后添加）

这部分是为了让软件更好用，但不是最紧急的。

### A. 历史管理（删除与回收）

#### 批量删除
- 在主窗口支持按住 Ctrl 多选路径，按 Delete 键删除。

#### 拖动删除
- 支持长按某条路径，拖动到窗口底部的"垃圾桶"区域进行删除。

#### 防误删机制（重要）
1. **删除操作仅仅是从 PathPilot 的数据库里删掉这条记录，绝对不允许调用系统的文件删除指令，不能删除用户硬盘上的真实文件。**
2. **提供"本地回收站"功能**：误删的记录保留24小时，支持 Ctrl+Z 撤销。
3. **活跃路径保护**：如果某条路径当前正在被资源管理器打开（状态为"活跃"），禁止删除该记录。

### B. 智能搜索与分组

#### 即时搜索
- 主窗口打开时，输入文字即可过滤列表中的路径。

#### 手动分组
- 允许用户右键某条路径，将其标记为"工作"、"个人"等分组，方便视觉归类。

### C. 隐私保护

#### 隐私模式
- 提供一个快捷键或托盘菜单选项。
- 开启后，软件暂停记录任何新的路径历史（适合处理敏感文件时）。

## 5. 明确不需要做的功能（避坑指南）

### 不要做
1. 文件预览、文件重命名、文件复制粘贴、解压压缩包。这些是资源管理器的事，我们不做。
2. 修改资源管理器的标题栏、不要在资源管理器里注入 DLL、不要做成浏览器标签页的形式。
3. 复杂的设置界面，所有配置尽量默认化或通过简单的 JSON 配置文件解决。

## 6. 技术约束与注意事项

### 系统兼容性
- 支持 Windows 10/11
- 不依赖特定版本的 Windows 组件

### 性能要求
- 后台运行时 CPU 占用 < 1%
- 内存占用 < 50MB
- 不影响系统资源管理器的正常使用

### 用户体验
- 启动时间 < 3 秒
- 路径记录延迟 < 500ms
- 界面响应时间 < 100ms

### 数据安全
- 本地存储，不上传云端
- 数据库文件可备份
- 支持隐私模式
