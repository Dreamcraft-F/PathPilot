## 📊 Tab职责

| Tab | 内容 | 说明 |
|-----|------|------|
| **目录**（默认） | 树形目录结构 | 按盘符和目录分组 |
| **时间** | 按时间分组 | 今日、昨日、本周、更早 |
| **频率** | 按频率分组 | 高频、中频、低频 |
| **★收藏** | 收藏路径 | 只显示收藏的路径 |
