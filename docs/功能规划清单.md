# PathPilot 后续拓展功能规划清单

## 🔴 P0 - 核心体验完善（建议立即修复）

| 功能 | 说明 | 当前状态 |
|------|------|----------|
| **路径删除功能** | 右键菜单删除单条路径记录 | 已实现 |
| **路径收藏功能** | 右键菜单收藏/取消收藏 | 已实现 |
| **路径跳转功能** | 点击路径打开资源管理器 | 已实现 |
| **界面样式优化** | 统一字体、颜色、间距 | 待优化 |

## 🟡 P1 - 体验增强（建议第二轮迭代）

| 功能 | 说明 | 优先级理由 |
|------|------|------------|
| **路径分组功能** | 按目录自动分组显示 | 提升浏览效率 |
| **访问时间显示** | 显示最后访问时间 | 便于识别近期路径 |
| **访问次数显示** | 显示访问次数统计 | 识别高频路径 |
| **快捷键支持** | Ctrl+C复制、Delete删除等 | 提升操作效率 |
| **窗口位置记忆** | 记住主窗口上次位置 | 提升用户体验 |
| **开机自启动** | 可选开机自动启动 | 便于常驻使用 |

## 🟢 P2 - 功能扩展（建议第三轮迭代）

| 功能 | 说明 | 优先级理由 |
|------|------|------------|
| **批量删除** | Ctrl+多选批量删除 | 大量数据管理 |
| **拖拽删除** | 长按拖动到垃圾桶区域 | 直观操作方式 |
| **路径标签** | 用户自定义标签（工作/个人等） | 个性化分类 |
| **导出/导入** | 导出历史记录为JSON/CSV | 数据备份迁移 |
| **路径统计** | 访问趋势图表、热力图 | 数据可视化 |
| **多显示器支持** | 悬浮球在多显示器间正确定位 | 多屏用户需求 |

## 🔵 P3 - 高级功能（长期规划）

| 功能 | 说明 | 优先级理由 |
|------|------|------------|
| **智能推荐** | 基于时间/地点推荐路径 | AI增强体验 |
| **跨设备同步** | 多台电脑间同步历史记录 | 多设备用户 |
| **插件系统** | 支持第三方插件扩展 | 生态建设 |
| **全局快捷键** | 系统级快捷键唤出主窗口 | 快速访问 |
| **路径预览** | 悬停显示路径内容摘要 | 快速识别 |
| **暗色模式** | 支持深色主题 | 视觉舒适度 |
