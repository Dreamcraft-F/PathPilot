## 功能概览

### 核心功能

| 模块 | 说明 |
|------|------|
| 无感路径捕获 | 窗口焦点监听 → 路径获取 → 过滤 → 去重 → 存储 |
| 系统托盘 | 圆形图标，隐私模式灰度，记录脉冲 |
| 悬浮球 | 纯圆形（Windows 11 DWM 修复），拖拽移动，点击打开主窗口 |
| 主窗口 | FluentWindow，左侧导航（目录/时间/设置/此电脑/清空/隐私），置顶 |
| 目录视图 | 树形结构，按盘符分组，收藏蓝色高亮，右键菜单 |
| 时间视图 | 按今日/昨日/本周/更早分组 |
| 路径跳转 | 激活已有窗口，找不到则新开 |
| 收藏/删除 | 通过信号通知 app 层处理业务逻辑 |
| 实时刷新 | 新路径捕获后 500ms 防抖刷新，跨线程信号安全 |
| 设置页面 | 开机自启动开关、导出功能（CSV/JSON）|
| 开机自启动 | 启动文件夹方式（创建快捷方式）|
| 首次运行向导 | 检测 data/.first_run_done，弹出欢迎页面 |
| 导出功能 | 支持 CSV 和 JSON 格式导出历史记录 |

## 技术栈

- **Python 3.11+**
- **PySide6** — GUI 框架
- **PySide6-Fluent-Widgets** — Fluent UI 风格组件
- **SQLite** — 本地数据存储
- **pywin32 / comtypes** — Windows API 调用
- **Nuitka** — 打包工具（standalone 模式）

## 项目结构

```
src/
├── main.py                 # 入口（首次运行检测、欢迎向导）
├── app.py                  # 应用主类，信号连接，业务逻辑
├── config/
│   └── config_manager.py   # 分层配置
├── core/
│   ├── window_monitor.py   # 窗口轮询监控（0.5s），Shell.COM 获取路径
│   ├── path_filter.py      # 策略模式过滤器
│   ├── path_dedup.py       # 时间+层级双层去重
│   ├── importance_calc.py  # 多维度重要性评分
│   └── path_engine.py      # 引擎协调器，事件队列+工作线程
├── database/
│   ├── models.py           # PathRecord, VisitEvent dataclass
│   └── db_manager.py       # SQLite，threading.Lock，软删除
├── gui/
│   ├── main_window.py      # FluentWindow，左侧导航
│   ├── tree_view.py        # QTreeWidget 树形目录
│   ├── time_view.py        # QTreeWidget 时间分组
│   ├── floating_ball.py    # 纯圆形悬浮球（DWM 修复）
│   ├── tray_icon.py        # 系统托盘
│   ├── settings_page.py    # 设置页面（自启动、导出）
│   ├── welcome_wizard.py   # 欢迎向导
│   └── icons.py            # 图标生成
├── resources/
│   ├── icon.png            # 应用图标
│   └── icon.ico            # exe 图标
└── utils/
    ├── logger.py           # 文件+控制台双输出（RotatingFileHandler）
    ├── path_utils.py       # 路径规范化
    ├── autostart.py        # 开机自启动（启动文件夹方式）
    └── export.py           # 导出功能（CSV/JSON）
```

## 关键技术细节

### Windows 11 悬浮球圆形修复

```python
# src/gui/floating_ball.py
ctypes.windll.dwmapi.DwmSetWindowAttribute(hwnd, 33, ctypes.byref(ctypes.c_int(1)), 4)  # 关闭圆角
ctypes.windll.dwmapi.DwmSetWindowAttribute(hwnd, 38, ctypes.byref(ctypes.c_int(1)), 4)  # 关闭系统背景
ctypes.windll.dwmapi.DwmSetWindowAttribute(hwnd, 34, ctypes.byref(ctypes.c_int(0)), 4)  # 去掉边框颜色
```

### Nuitka 打包后路径检测

```python
# src/main.py
BASE_DIR = os.path.dirname(os.path.abspath(sys.argv[0]))
```

### 开机自启动实现

```python
# src/utils/autostart.py
startup_folder = os.path.join(os.environ['APPDATA'], r'Microsoft\Windows\Start Menu\Programs\Startup')
```

## 已知限制

1. 仅支持 Windows — 使用了 Win32 API 和 COM 接口
2. Windows 11 特定修复 — DWM API 仅在 Windows 11 上需要
3. 打包体积较大 — PySide6 本身体积大
4. 无自动更新 — 需要用户手动下载新版本
